## Attributes

| 参数   | 说明               | 类型          | 可选值 | 默认值 |
| ------ | ------------------ | ------------- | ------ | ------ |
| name   | 图标名称(svg)      | string        | –      | –      |
| icon   | 图标名称(iconfont) | string        | –      | –      |
| size   | 图标大小           | number/string | –      | –      |
| rotate | 图标是否持续旋转   | boolean       | –      | false  |
